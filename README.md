# server-for-jzx

> 安装 koa2 脚手架

```
npm install -g koa-generator
```

> 该项目使用 mongodb 作为数据库

[安装 mongodb 教程](https://www.runoob.com/mongodb/mongodb-osx-install.html)

```
sudo brew install mongodb
sudo mkdir -p /data/dbsudo mongod

# 如果没有创建全局路径 PATH，需要进入以下目录
cd /usr/local/mongodb/bin
sudo ./mongod

```

> 安装

```
npm install
```

> 运行

```
npm start
```

> h5Editor 每次更新完 component-for-jzx 需要执行命令 npm run build 来更新 ssr 的模板页面
